# KellyCash PRO

Aplicación de finanzas familiares inspirada en las hojas Excel actuales.

Objetivos:
- Gestión de ingresos
- Gestión de gastos compartidos e individuales
- Gestión de deudas
- Gestión de ahorro
- Objetivos financieros
- Patrimonio familiar
- Reportes
- Adjuntos

Stack:
- Next.js
- TypeScript
- Supabase
- Tailwind
- Shadcn UI
