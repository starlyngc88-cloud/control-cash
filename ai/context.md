# Proyecto
 
Family Finance Manager
 
Aplicación web para gestión financiera familiar.
 
Objetivo:
 
- Controlar ingresos
- Controlar gastos
- Controlar deudas
- Controlar ahorros
- Gestionar objetivos financieros
 
Tecnologías:
 
- Next.js
- TypeScript
- Supabase
- Tailwind
- Shadcn UI
 
Restricciones:
 
- Coste cero
- Uso personal
- Sin AWS
- Sin Docker
- Sin microservicios
